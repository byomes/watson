Different Greek Texts |  |  | there are three main Greek texts that are used for Bible Translation today. Here is an overview of them
 |  |  | 1. The Textus Receptus
 |  |  |  (Latin for "Received Text"), this is the work of a 15th century Dutch theologian named Erasmus. He was racing
 |  |  | with a competing group in Spain to be the first to write a full Greek manuscript of the Scriptures. Having only a few documents
 |  |  | to work with, and a deadline to beat, Erasmus's orginal Greek Text was first published in 1516, with updated and corrected 
 |  |  | additions coming later in 1519, 1527, and 1535. The Textus Receptus was the first Greek text to be used to make vernacluar
 |  |  | translations for the common people of the day. The KJV, NKJV, MEV, and Reformation-era Bibles are based on this text.
 |  |  | 2. The Majority Text 
 |  |  | The majority text is a Greek manuscript that is often called the "democratic text." With the Scriptures being copied over the 
 |  |  | centuries, scribal errors crept their way in. The Majority text views all the manuscripts as equal and essentially gives each one
 |  |  | a single vote as to which interpretation should be accepted. There are no English tranlsations based on this text. 
 |  |  | 3. The Critical Text 
 |  |  | The Critical text is a Greek manuscript that is made from a group of ancient manuscripts and their variants
 |  |  | in an attempt to preserve the original writing of Scripture. Unlike the Majority Text, the Critical takes into account the age
 |  |  | of a document as well as the school or family from which that document arises. This is done through a process called
 |  |  | textual criticism. This text is viewed by virtually all Biblical scholars as the most accurate Greek text to date. All English
 |  |  | translations of the Bible other than those based on the Textus Recptus are based on this text.
